# Contributing to KitroJS

Thank you for considering contributing to KitroJS!

## How to contribute

1. Fork the repository

2. Create a new branch:

   git checkout -b feature/my-feature

3. Commit your changes:

   git commit -m "Add my new feature"

4. Push your branch:

   git push origin feature/my-feature

5. Create a Pull Request

## Code Style

- Use clean, modern JavaScript / TypeScript

- Follow the structure of existing blocks and modules

- Keep UI consistent with the Kitro design language

## Issues & Bugs

If you discover a bug, please open an issue with:

- steps to reproduce

- screenshots/logs if possible

- expected vs actual behavior

## Security

If you find a security issue, **do NOT create a public issue**.

Email: security@kitrojs.com
